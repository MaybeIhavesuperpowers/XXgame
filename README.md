# 像素纪元：五域远征

零依赖 HTML5 Canvas ARPG。包含 5 个大陆、每域 5 层程序化地图、即时战斗、Boss、资源采集、随机装备、主动战技、天赋星盘、营地铁匠铺和多档位本地存档。角色、20 种普通怪物、5 名 Boss、场景物件、地形、装备、材料与技能特效均使用独立美术图集，并带有待机动画或环境动态效果。

角色换装采用 JSON 驱动的分层挂载架构：身体、头发、头盔、护甲、武器、护盾和特效分别排序，装备绑定逐帧 `head/body/leftHand/rightHand/back` 锚点并继承角色位置、方向、动画状态与帧号。架构说明见 [`docs/equipment-system.md`](docs/equipment-system.md)，最小示例位于 [`demo/equipment-system.html`](demo/equipment-system.html)。

角色采用头、双手、身体、双脚和武器五部分模块化换装渲染，各部件会跟随移动和攻击独立运动；步行使用 2 帧循环，双击方向触发的奔跑使用 4 帧循环。背包容量为 120 格，可翻 4 页、对比同部位装备并按品质批量出售。

五类武器拥有不同节奏与判定：长剑三连斩、长枪贯穿突刺、战斧裂地重劈、星杖弹体攻击、双刃疾影连刺。普通怪物各有专属攻击，Boss 每名拥有 4 套招式并在低生命阶段强化。

怪物装备掉落会独立随机品质、基础数值、基础词条和特殊词条。特殊词条范围包括元素增伤 4%–14%、异常触发 2%–12%、斩杀增伤 5%–15%、精英增伤 6%–18%、弹体威力 8%–25% 与稀有寻获 5%–20%。

采集材料可打造装备、强化和炼制药水；怪物专属稀材可作为打造催化剂，Boss 核心用于稀有品质保底，虚空资源用于锁定词条洗练。

音频采用 Web Audio 原创程序化生成：标题、营地以及 5 个大陆的 25 个深度均有名称固定、常驻循环的独立 BGM，同一大陆的五层会改变旋律走向、节奏、和声与环境音色；武器攻击、怪物招式、Boss 技能、命中、受伤、掉落、锻造、传送与保存均使用独立音效通道。受浏览器自动播放策略限制，首次点击或按键后开始播放。

## 启动

双击 `start.ps1`，或在项目目录运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\start.ps1
```

随后打开 `http://127.0.0.1:4173`。分层装备系统通过 JSON 加载配置，因此请使用本地服务器启动，不再建议直接双击 `index.html`。

## 上线验收

无需安装依赖即可运行资源、配置、动画帧、装备锚点、DOM、20 种怪物招式、20 套 Boss 招式和 25 首关卡 BGM 的完整性检查：

```powershell
npm test
```

本地服务器下访问 `http://127.0.0.1:4173/?qa=1` 会显示仅限本机的验收面板，可快速生成装备/材料样本、切换五类武器、进入任意深度、触发怪物与 Boss 特效、检查三类掉落以及跑完整的走路/奔跑/攻击/受击动画。普通线上地址不会显示此面板。

## 操作

- `WASD`：行走；快速双击并按住某个方向进入奔跑
- 鼠标左键 / `J`：360° 瞄准的武器三段攻击
- `R`：当前武器的专属主动战技
- `F`：星爆术
- `空格`：闪避
- `Q`：生命药水
- `E`：拾取地面装备/药水/材料、采集、点亮祭坛、进入传送门
- `C` / `I`：角色与背包（也可点击左上角常驻“背包”按钮）
- `Tab`：远征区域观测地图；按钮或鼠标滚轮可在全图与局部视野间缩放
- `K`：主动战技说明与被动天赋星盘
- `B`：铁匠铺（仅营地可用）
- `M`：星门航图（仅营地可用）
- `T`：从远征随时返回星火营地
- `L`：打开自动存档与三个手动存档的列表
- `F5` / `Ctrl+S`：立即写入自动存档
- `Esc`：暂停或关闭面板

存档写入浏览器 `localStorage`，每 30 秒、过层、打造/强化和返回营地时自动保存。存档会记录当前地图、玩家位置、存活怪物、采集点、地面掉落和祭坛状态；主存档校验失败时会自动读取上一份备份，并兼容迁移旧版存档。
